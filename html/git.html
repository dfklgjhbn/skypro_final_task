Git

  • git add <файл> — выбирает конкретный файл для включения в будущий коммит, добавляет его в индекс.
  • git commit -m «Описание изменений» — формирует новый коммит с комментарием, описывающим суть внесённых правок
  • git branch — отображает перечень всех локальных веток проекта. git branch <название> — создаёт новую ветку, оставаясь при этом в текущей.
  • git checkout <ветка> — осуществляет переход на указанную ветку. git checkout -b <ветка> — создаёт новую ветку и сразу же переключается на неё.
  • git merge <ветка> — объединяет выбранную ветку с текущей, добавляя её изменения.
